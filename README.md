# Taller: Fundamentos en análisis de información y minería de datos en agricultura específica por sitio

El taller desarrollado del 28 al 30 de diciembre en Lima Perú, consistió en mostrar los conceptos y herramientas en análisis de datos priorizando el uso del software estadístico libre R. Incluyendo un refuerzo de conceptos teóricos y también una serie de ejercicios prácticos. El material de las carpetas se ha sido distribuido de la siguiente manera:

* La primera carpeta (Presentaciones Generales) está compuesta por dos archivos, uno que habla de conceptos básicos de agricultura digital y minería de datos y otra que habla sobre la obtención de datos en distintas fuentes de información.

* La segunda carpeta (Introducción a R y análisis exploratorio) contiene tres presentaciones, una de ellas, para introducir al grupo el software R, otra un repaso del análisis descriptivo, y una de gráficos descriptivos en R. Adicionalmente, por cada tema, se dejaron tres de los scripts utilizados como ejemplos desarrollados en conjunto con los participantes en el mismo taller. También se encontrará un documento donde se han propuesto ejercicios prácticos para los participantes, y en adjunto un script que es la solución a dicho taller.

* La tercera carpeta (Procesamiento de datos y minería de datos) contiene tres presentaciones, la primera muestra las estrategias de procesamiento que usualmente se implementan sobre bases de datos de distintas estructuras, la segunda consiste en mostrar los casos más comunes de errores en la base de datos y la manera en las que se toman acciones, y la tercera, una introducción a la minería de datos. Cada sección está vinculada a uno o varios scripts para los ejemplos prácticos.

* La cuarta carpeta (Regresión y Tablue), con contenido de Regresión Lineal Múltiple, análisis de varianza y material relacionado del software Tableu para análisis de datos.

Finalmente se adjunta una carpeta aparte (Datos), con bases de datos de prueba que fueron utilizadas en los diferentes ejercicios del taller.
